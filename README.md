# kali-desktop-kasm


Based on `kasmweb/kali-rolling-desktop:1.17.0-rolling-weekly`.

Custom Kali Desktop image for Kasm Web that installs the following additional packages:

```
ffuf
dirb
seclists
gobuster
hashcat
golang
exploitdb
ssh
freerdp3-x11
openvpn
zsh-autosuggestions
pipx
powershell
evil-winrm
dnsrecon
feroxbuster
onesixtyone
oscanner
sipvicious
smbmap
tnscmd10g
dnsutils
code-oss
iputils-ping
dnsenum
ftp
sqsh
eyewitness
php
socat
tmux
```

The image rebuilds weekly after the Kasm Web weekly rolling image has been released.
